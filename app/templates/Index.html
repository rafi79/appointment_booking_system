API Endpoints

Authentication

POST /api/v1/auth/register
Register new user (patient/doctor)
POST /api/v1/auth/login
User login with JWT token
GET /api/v1/auth/profile
Get current user profile

Doctors

GET /api/v1/doctors/
List all doctors with filters
GET /api/v1/doctors/search
Search doctors by name/specialization
GET /api/v1/doctors/{id}
Get doctor details by ID

Appointments

POST /api/v1/appointments/
Create new appointment
GET /api/v1/appointments/
Get user's appointments
PUT /api/v1/appointments/{id}/status
Update appointment status

Admin

GET /api/v1/admin/dashboard
Admin dashboard statistics
GET /api/v1/admin/reports/monthly
Generate monthly reports
GET /api/v1/admin/users
Manage all users

Setup Instructions:
  1. Install dependencies: pip install -r requirements.txt
  2. Set up environment variables: cp .env.example .env
  3. Run database migrations: python scripts/seed_data.py
  4. Start the server: uvicorn app.main:app --reload
  5. Visit http://localhost:8000 to access the API
